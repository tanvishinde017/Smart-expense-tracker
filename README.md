💰 Smart Expense Tracker
📌 Overview

Smart Expense Tracker is a full-stack web application developed as a college project to help users efficiently manage and monitor their daily expenses. The system allows users to record transactions, categorize spending, and track overall expenses, promoting better financial discipline through structured budgeting.

This project showcases practical implementation of backend development, database integration, and web application design principles.

🎯 Objective

The key objectives of this project include:

Applying web development concepts in a real-world use case
Building a functional personal finance management system
Understanding and implementing CRUD operations
Gaining experience in backend development and database handling
Developing problem-solving and structured project design skills

✨ Key Features
Add and manage daily expense entries
Categorize transactions (Food, Transport, Bills, Shopping, etc.)
View complete transaction history
Track total spending dynamically
Budget monitoring functionality
Edit and delete transactions
Secure user authentication system
Persistent storage using a database
Clean, responsive, and user-friendly interface
